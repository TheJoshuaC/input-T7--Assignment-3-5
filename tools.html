Tools & Technology

GitHub – GitHub will be used to store and manage our code, while also keeping track of any changes that are made to the code along the way while designing the website. The team has all had experience using this program in the past some more than others, so we shouldn’t have many dramas when it comes to uploading our code.

Python IDLE – Python will be the main coding language we will use while designing this project. Using IDLE will help in the process of writing our code more efficiently and provides a built-in debugger for troubleshooting any errors along the way. The teams prior experience with Python does vary, only a few of our members have experience in using this and they will be the ones that will focus on writing up the code.

Atom/Sublime/Notepad++ –These three are the text editor programs that we will be using to write up all our code needed for the website. All members of the team have prior experience using at least one of these three programs, but these will be used hand in hand with the coding process. Microsoft PowerPoint – We will be using Microsoft PowerPoint to make our presentation on our project. The team has had prior experience in using this program, however some of the members are more capable of using it to a higher standard than others and will be best suited to construct the presentation or overview the finished product.

Adobe Photoshop – Adobe Photoshop is the program we have decided to use for the graphic designs for our website. Not all the team members have prior experience using this program, with that in mind the members with experience will be handling the graphic design aspect of the project.
